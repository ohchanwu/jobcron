# job-scraper

**신입 IT Job Briefing** — a calm daily job-posting briefing for Korean
new-grad (신입) IT job seekers.

`job-scraper` is a single binary that opens a local web app. Click "스크랩
시작" and it scrapes 점핏 (Jumpit), scores new-grad IT postings against your
profile, and shows a one-page daily briefing — every match explained, no
notifications, no setup.

> **Status: in active development (v1).** This README is a placeholder; a
> proper overview, screenshots, and per-platform install instructions land
> before the first release.
